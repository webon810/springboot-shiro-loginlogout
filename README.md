# springboot-shiro-loginlogout
springboot-shiro-loginlogout

## What you'll need
- JDK 1.8+
- Maven 3+

## Stack
- Java (SE8)
- Spring Boot (v1.5.10)
- FreeMarker
- Shiro

## Run
- Run Authentication Service: mvn spring-boot:run

## Testing by using following urls
localhost:8888/index
localhost:8888/login
localhost:8888/logout
localhost:8888/secure
